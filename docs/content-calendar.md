# Unvibe Content Calendar — MVP Phase

## 🎯 Core Series: "The Reframe"

**Hook:** Playing games = "fighting mental debt caused by agentic development" — the irony sells the value.

This series reframes guilt-free rest (gaming, breaks, walks) as *productive recovery* — not laziness, not slack, but strategic cognitive maintenance that enables sustainable high-output dev work.
