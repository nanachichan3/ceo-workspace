# Self-Degree Landing Page

**Self-Degree Framework** — Landing page MVP.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm run dev    # Development
npm run build  # Production build
npm start      # Production server
```

## Deployment

Deployed to Vercel. Point domain to `vercel.com` or connect repo directly.
